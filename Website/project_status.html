    Status Update - Week of 11/13

    Problems
  • Rain is interfering with testing
  • IMU and Temperature sensors were removed from the design due to difficulties communicating and incorrect data respectively
    • Temperature is now being calculated on the pressure sensors
    •  Current Status
  • Most of the code has been tested and is working as anticipated
  • Enclosure has been redesigned to fit within weight limits - maximize ballast capacity
  • Communication with the IMU has not been solved. It is thought to be an issue with the TxRx ports as communication via USB works. This connection is not feasible for the launch though. Kyle has run the simulations, and taking the IMU out of the design will not have a significant impact on the control algorithm.
    •  Task Status
  • The group tested most systems this past weekend. Pressure sensors, GPS, and temperature measurements seem to be working. Control of the servos for helium and ballast release is working. The cut-down method and code is working at ground temperature and conditions.
  • Communications code was fixed for sending messages from the payload to the ground station.
    •  Planned Tasks
  • Test sending messages from ground station to payload
  • Test altitude change measurement (start from ground, move to roof of building and check altitude)
  • Full dry run of control code (try to release ballast and "vent" helium with altitude change)
  • Dry run of setup for launch
    •
