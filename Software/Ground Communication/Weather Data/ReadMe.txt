%% Weather Data %%

Get weather data from NOAA online at: 
ftp://nomads.ncdc.noaa.gov/GFS/Grid4

Dowlnoad the *.grb2 files asssociated with the time of interest

On a linux machine use the WGRIB2 software available here:
http://www.cpc.ncep.noaa.gov/products/wesley/wgrib2/

Follw the instruction written in 'writecsv.sh' to create *.txt files able to be used by the matlab simulation script 

'gfs_3_20180709_0000_000.txt' is an example file showing the expected format and naming convention
